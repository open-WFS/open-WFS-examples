# OpenWFS: Example audio projects 

- [Reaper Broadcast 1-8](Reaper%20Broadcast%201-8): Repeatedly and simultaneously speaks the numbers one to eight, routed such that "one" should be heard through every channel of Module 1, and so forth
- [Reaper Speech Example 4ch](Reaper%20Speech%20Example%204ch): Example containing four long mono speech segments, in different languages, which can be routed to Spat as mono channels to spatialise and test speech intelligibility
